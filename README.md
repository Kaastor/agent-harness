# Agentic Systems Laboratory

This repository contains three separate agentic-system projects. Each project
has its own boundary, research question, and `IDEA.md`.

## Systems

- [`agentqual/`](agentqual/IDEA.md) — qualifies whether a candidate model can
  replace the reference model in a concrete agentic system.
- [`laboratory-class-harness/`](laboratory-class-harness/IDEA.md) — a working Pi
  harness for controlled, evidence-producing agent experiments. Its source,
  tests, examples, and package configuration live in that directory.
- [`agent-tool-control-plane/`](agent-tool-control-plane/IDEA.md) — controls and
  reconciles agent-initiated tool operations with external side effects.

The projects may exchange evidence, but they remain separate systems rather
than modules of one prematurely unified platform.
